03. FURNITURE DESIGN

 A.Untitled

This project is called "Untitled" because it does not have a name. "Untitled" contains 04 kitchen styles and 06 house furniture collections, created from 2021 to 2022. My role is to make the concepts, renderings, drawings, write technical data and systemize them so that these furniture can be used in a software (or an app). The purpose is to optimize the interior design process by providing the "ready-made" furniture which can be customize some features such as dimension and color. During the design, I learn about interior materials (mostly industrial wood), the CNC cutting process and technical drawings.

Though "Untitled" has never been finished, I am proud to be a part of it.
